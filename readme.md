## 🪄 Customer Value Projection Tool by Andrei Malang
*In compliance with Finals Requirements for the course Machine Learning [CS307] under Sir Obien.*


A Machine Learning dashboard that predicts a customer's **Spending Score** based on demographic and behavioral data. This tool is designed to help businesses identify high value engagement patterns while being transparent with how the model handles and prioritizes each factor.

---

## 📊 Dataset Source
The model is trained on the **Customer Segmentation Dataset** available on Kaggle.
* **Source:** [Kaggle - Customer Segmentation Data](https://www.kaggle.com/datasets/fahmidachowdhury/customer-segmentation-data-for-marketing-analysis)
* **Features Used:** Age, Gender, Annual Income, Membership Years, Purchase Frequency, and Preferred Category.
* **Features Predicted:** Spending Score
